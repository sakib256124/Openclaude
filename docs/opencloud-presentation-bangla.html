OpenCloud Compute ConsoleSlide 1

Project Overview

এই প্রজেক্টের মূল উদ্দেশ্য

Problem

Ubuntu Multipass VM manage করার জন্য সাধারণ user-friendly web console দরকার।

Solution

একটি dashboard যেখানে instance, image, network, volume, security, metrics ইত্যাদি এক জায়গা থেকে দেখা যাবে।

Target User

Cloud admin, project user, viewer এবং educational lab environment.

Important Point

এটি AWS নয়; UI inspiration EC2-style, কিন্তু backend target Multipass CLI.

Project OverviewSlide 2

Technology Stack

কোন কোন technology ব্যবহার করা হয়েছে

Frontend

Next.js App Router, React, TypeScript, Tailwind CSS, Radix UI, Lucide icons.

Backend

Next.js API routes, server-only Multipass service layer, Zod validation.

Database

PostgreSQL database এবং Prisma ORM দিয়ে schema, migration, seed manage করা হয়েছে।

Authentication

Auth.js / NextAuth credentials login, JWT session এবং bcrypt password hashing.

Testing

Vitest unit test, Playwright end-to-end testing setup.

Deployment

Docker, Docker Compose এবং Nginx reverse proxy configuration.

Multipass ServicesSlide 5

Implemented Features

বর্তমানে project-এ যা কাজ করছে

  • Secure login system with Auth.js credentials provider.
  • User management: create, update, role change, password reset.
  • Role-based access control: ADMIN, USER, VIEWER.
  • Application settings, user preferences, notifications এবং activity logs.
  • Dashboard shell, sidebar navigation, responsive UI states.
  • Instance API Multipass CLI call করে; CLI unavailable হলে safe local fallback নেয়।
  • Images, key pairs, networks, security groups, volumes, snapshots, billing এবং search API-backed.

AWS Concept CoverageSlide 7

Security

Security design

Password

Password plain text রাখা হয় না; bcrypt hash store করা হয়।

Credential

Multipass host access setting server-side থাকে এবং AES-256-GCM দিয়ে encrypt করার helper আছে।

RBAC

প্রতিটি sensitive API route permission check করে।

Headers

CSP, X-Frame-Options, nosniff, Referrer-Policy middleware থেকে set হয়।

Validation

Zod schema দিয়ে request payload validate করা হয়।

Audit Log

Login, user create/update, settings change safe audit message সহ record হয়।

Current Status

বর্তমান limitation এবং honest status

  • Authentication, user-role, settings, logs এবং UI foundation ready.
  • Instance list, launch, action এবং details Multipass CLI try করে; CLI না থাকলে local fallback নেয়।
  • Cloud-like resources database/control-plane level-এ API-backed.
  • AWS-only concepts যেমন true Elastic IP routing বা EBS block device host automation ছাড়া limited.

এই design real infrastructure না থাকলেও app crash না করে honest fallback এবং clear limitation দেখায়।
